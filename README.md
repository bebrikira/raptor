# Crypto Portfolio Mock

A lightweight cryptocurrency portfolio tracker built with Python.

## Overview

Crypto Portfolio Mock is a simple application that simulates cryptocurrency portfolio management. It allows users to track assets, monitor portfolio value, and calculate profit or loss using mock market data.

This project was created as a practice application to demonstrate Python fundamentals, clean project structure, and API-ready architecture.

## Features

- Portfolio balance overview
- Asset tracking
- Profit/Loss calculation
- Mock cryptocurrency prices
- Simple and readable codebase
- Easy to extend with real APIs
